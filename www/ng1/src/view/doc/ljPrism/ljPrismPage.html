ljPrism指令

该指令一般不使用，大多数都是直接使用没封装的形式

由于{ {}}会被ng解析，所以在第一个花括号后面加一个空格来破坏， 这样{ {}}就可以直接当字面量来显示了

效果预览

数据格式 

                
                    $scope.model = {
                        language: 'language-css',
                        source: '.p-ico-arrow.next{background-position:0 -198px;}',
                    };

使用方式 

                
                    <lj-prism language="{ {model.language}}" source="{ {model.source}}"></lj-prism>

参数说明

没封装的使用方式 

                
                    <pre>
                        <code class="{ {language} }">
                            { {source} }
                        </code>
                    </pre>
